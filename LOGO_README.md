# UNHCR Logo

To use the official UNHCR logo:

1. Download the UNHCR logo from the official UNHCR brand guidelines
2. Save it as `unhcr-logo.png` or `unhcr-logo.svg` in this directory
3. Replace the SVG in `index.html` with:

```html
<img src="unhcr-logo.png" alt="UNHCR Logo" class="unhcr-logo">
```

Or update the CSS to use the image file instead of the SVG placeholder.
